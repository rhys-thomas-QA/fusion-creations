# Fusion Creations ad creatives

Short Remotion video templates for Fusion Creations marketing.

## Current compositions

- `FusionBadgeAdVertical`: 1080 x 1920, 15 seconds
- `FusionBadgeAdSquare`: 1080 x 1080, 15 seconds
- `FusionBadgeAdLandscape`: 1920 x 1080, 15 seconds
- `ClearScoreProofAdVertical`: 1080 x 1920, 15 seconds
- `ClearScoreProofAdSquare`: 1080 x 1080, 15 seconds
- `ClearScoreProofAdLandscape`: 1920 x 1080, 15 seconds

Each candidate uses one React component across three formats, so changes can be tested quickly.

## Commands

```console
npm install
npm run dev
npm run lint
npm run still:vertical
npm run still:proof:vertical
npm run render:vertical
npm run render:square
npm run render:landscape
npm run render:proof:vertical
npm run render:proof:square
npm run render:proof:landscape
```

Rendered files go into `out/`, which is ignored by Git.
