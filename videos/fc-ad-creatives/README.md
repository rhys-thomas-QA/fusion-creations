# Fusion Creations ad creatives

Short Remotion video templates for Fusion Creations marketing.

## Current compositions

- `FusionBadgeAdVertical`: 1080 x 1920, 15 seconds
- `FusionBadgeAdSquare`: 1080 x 1080, 15 seconds
- `FusionBadgeAdLandscape`: 1920 x 1080, 15 seconds

All three use the same React component and copy, so changes can be tested across formats quickly.

## Commands

```console
npm install
npm run dev
npm run lint
npm run still:vertical
npm run render:vertical
npm run render:square
npm run render:landscape
```

Rendered files go into `out/`, which is ignored by Git.
